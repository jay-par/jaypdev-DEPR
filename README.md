## Development

Running locally
```bash
npm run dev
```

Building
```bash
npm run build
```

Starting
```bash
npm run start
```

Deployment in production mode
Building
```bash
npm run deploy
```

Tech stack includes:
- NextJS
- Styled components & icons
- Netlify
